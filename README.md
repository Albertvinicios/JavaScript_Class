# 📘 JavaScript Class

Repositório de estudos baseado no curso de JavaScript do Curso em Vídeo, com resoluções próprias dos exercícios, comentários explicativos e organização voltada para aprendizado prático.

---

## 🚀 Objetivo

Demonstrar minha evolução em JavaScript através da resolução de exercícios, destacando:

* Minha lógica de programação
* Diferentes formas de resolver problemas
* Explicações comentadas para facilitar o entendimento

Este repositório não mostra apenas o resultado final, mas o **processo de pensamento por trás das soluções**.

---

## 🧠 Metodologia

Cada exercício foi resolvido:

* Com abordagem própria (não apenas cópia do curso)
* Com comentários explicativos no código
* Pensando em quem está aprendendo também

A ideia é que qualquer pessoa consiga **aprender junto lendo o código**.

## ▶️ Como executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/javascript-class.git
```

2. Acesse a pasta do projeto:

```bash
cd javascript-class
```

3. Abra o arquivo `index.html` no navegador.
